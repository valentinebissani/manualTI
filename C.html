Tutorial de Linguagem C

O que é Linguagem C?

C é uma linguagem de programação estruturada, criada por Dennis Ritchie em 1972 nos laboratórios Bell. É amplamente usada para:

  • Desenvolvimento de sistemas operacionais
  • Programação de microcontroladores
  • Desenvolvimento de drivers
  • Sistemas embarcados
  • Núcleo de outras linguagens

Estrutura Básica de um Programa C

                
#include <stdio.h>

int main() {
    // Seu código aqui
    printf("Olá, Mundo!\n");
    return 0;
}

Explicação:

  • #include <stdio.h> - Inclui a biblioteca padrão de entrada/saída
  • int main() - Função principal onde a execução começa
  • printf() - Função para imprimir na tela
  • return 0 - Indica que o programa terminou com sucesso
  • // - Comentário de uma linha

Tipos de Dados Básicos

Tipo Descrição
int Números inteiros (normalmente 4 bytes)
float Números decimais (4 bytes)
double Números decimais com maior precisão (8 bytes)
char Um único caractere (1 byte)
void Tipo vazio (usado em funções sem retorno)

Variáveis e Constantes

Declaração de variáveis:

int idade = 25;
float altura = 1.75f;
char inicial = 'J';
int programador = 1; // 1 para true, 0 para false

Constantes (valores que não mudam):

#define PI 3.14159
const int DIAS_SEMANA = 7;

Operadores

Operadores Aritméticos

  • + Adição
  • - Subtração
  • * Multiplicação
  • / Divisão
  • % Módulo (resto da divisão)

Operadores de Comparação

  • == Igual a
  • != Diferente de
  • > Maior que
  • < Menor que
  • >= Maior ou igual a
  • <= Menor ou igual a

Condicionais

if-else:

if (idade >= 18) {
    printf("Maior de idade\n");
} else {
    printf("Menor de idade\n");
}

switch-case:

switch(dia) {
    case 1: printf("Domingo\n"); break;
    case 2: printf("Segunda\n"); break;
    // ...
    default: printf("Dia inválido\n");
}

Loops

for:

for(int i = 0; i < 10; i++) {
    printf("%d\n", i);
}

while:

int i = 0;
while(i < 10) {
    printf("%d\n", i);
    i++;
}

do-while:

int i = 0;
do {
    printf("%d\n", i);
    i++;
} while(i < 10);

Funções

Funções permitem organizar e reutilizar código:

// Declaração
int soma(int a, int b) {
    return a + b;
}

// Chamada
int resultado = soma(5, 3);
printf("5 + 3 = %d\n", resultado);

Arrays (Vetores)

Armazenam múltiplos valores do mesmo tipo:

int numeros[5] = {10, 20, 30, 40, 50};

// Acessando elementos
printf("%d\n", numeros[0]); // Primeiro elemento (10)

// Alterando valores
numeros[1] = 25;

Ponteiros

Um dos conceitos mais importantes em C:

int var = 10;
int *ptr = &var; // ptr armazena o endereço de var

printf("Valor: %d\n", *ptr); // Acessa o valor através do ponteiro
*ptr = 20; // Modifica o valor através do ponteiro
