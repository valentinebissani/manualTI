Tutorial de Java

O que é Java?

Java é uma linguagem de programação orientada a objetos, criada pela Sun Microsystems em 1995 (atualmente pertencente à Oracle). É amplamente usada para:

  • Desenvolvimento de aplicativos Android
  • Aplicações empresariais
  • Sistemas bancários
  • Big Data (Hadoop, Spark)
  • Internet das Coisas (IoT)

Estrutura Básica de um Programa Java

                
public class Main {
    public static void main(String[] args) {
        // Seu código aqui
        System.out.println("Olá, Mundo!");
    }
}

Explicação:

  • public class Main - Define uma classe pública (o nome do arquivo deve ser Main.java)
  • public static void main(String[] args) - Método principal onde a execução começa
  • System.out.println() - Para imprimir na tela
  • // - Comentário de uma linha

Tipos de Dados Básicos

Tipo Descrição
int Números inteiros
double Números decimais
float Números decimais

Variáveis e Constantes

Declaração de variáveis:

int idade = 25;
double altura = 1.75;
char inicial = 'J';
boolean programador = true;

Constantes (valores que não mudam):

final float PI = 3.14159f;
final int DIAS_SEMANA = 7;

Operadores

Operadores Aritméticos

  • + Adição
  • - Subtração
  • * Multiplicação
  • / Divisão
  • % Módulo (resto da divisão)

Operadores de Comparação

  • == Igual a
  • != Diferente de
  • > Maior que
  • < Menor que
  • >= Maior ou igual a
  • <= Menor ou igual a

Condicionais

if-else:

if (idade >= 18) {
    System.out.println("Maior de idade");
} else {
    System.out.println("Menor de idade");
}

switch-case:

switch(dia) {
    case 1: System.out.println("Domingo"); break;
    case 2: System.out.println("Segunda"); break;
    // ...
    default: System.out.println("Dia inválido");
}

Loops

for:

for(int i = 0; i < 10; i++) {
    System.out.println(i);
}

while:

int i = 0;
while(i < 10) {
    System.out.println(i);
    i++;
}

do-while:

int i = 0;
do {
    System.out.println(i);
    i++;
} while(i < 10);

Métodos

Métodos permitem organizar e reutilizar código:

// Declaração
public static int soma(int a, int b) {
    return a + b;
}

// Chamada
int resultado = soma(5, 3);
System.out.println("5 + 3 = " + resultado);

Arrays

Armazenam múltiplos valores do mesmo tipo:

int[] numeros = {10, 20, 30, 40, 50};

// Acessando elementos
System.out.println(numeros[0]); // Primeiro elemento (10)

// Alterando valores
numeros[1] = 25;

Programação Orientada a Objetos

Java é uma linguagem totalmente orientada a objetos:

public class Carro {
    // Atributos
    private String marca;
    private int ano;

    // Construtor
    public Carro(String m, int a) {
        marca = m;
        ano = a;
    }

    // Método
    public void mostrarInfo() {
        System.out.println("Marca: " + marca + ", Ano: " + ano);
    }
}

// Usando a classe
Carro meuCarro = new Carro("Ford", 2020);
meuCarro.mostrarInfo();
