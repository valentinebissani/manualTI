Tutorial de JavaScript

O que é JavaScript?

JavaScript é uma linguagem de programação interpretada, criada por Brendan Eich em 1995. É a principal linguagem para desenvolvimento web e é usada para:

  • Desenvolvimento front-end (junto com HTML/CSS)
  • Desenvolvimento back-end (Node.js)
  • Aplicações móveis (React Native, Ionic)
  • Jogos (com HTML5 Canvas)
  • Automação de tarefas

Estrutura Básica de um Programa JavaScript

// Comentário de uma linha
/*
  Comentário de múltiplas linhas
*/

// Exemplo simples
console.log("Olá, Mundo!");

Tipos de Dados Básicos

Tipo Descrição
Number Números inteiros ou decimais
String Texto (entre aspas simples ou duplas)
Boolean Verdadeiro (true) ou falso (false)
Object Coleção de pares chave-valor
Array Lista ordenada de valores

Variáveis e Constantes

Declaração de variáveis (ES6+):

let idade = 25;          // Variável mutável
const PI = 3.14159;     // Constante (não pode ser reatribuída)
var nome = "João";      // Forma antiga (evitar usar)

Operadores

Operadores Aritméticos

  • + Adição
  • - Subtração
  • * Multiplicação
  • / Divisão
  • % Módulo (resto da divisão)
  • ** Exponenciação (ES6)

Operadores de Comparação

  • == Igualdade (com conversão de tipo)
  • === Igualdade estrita (sem conversão)
  • != Diferente
  • !== Diferente estrito
  • > Maior que
  • < Menor que

Estruturas de Controle

Condicionais

if-else:

if (idade >= 18) {
    console.log("Maior de idade");
} else {
    console.log("Menor de idade");
}

Operador ternário:

const status = idade >= 18 ? "Adulto" : "Menor";

switch-case:

switch(dia) {
    case 1: console.log("Domingo"); break;
    case 2: console.log("Segunda"); break;
    default: console.log("Dia inválido");
}

Loops

for:

for(let i = 0; i < 10; i++) {
    console.log(i);
}

while:

let i = 0;
while(i < 10) {
    console.log(i);
    i++;
}

for...of (ES6):

const frutas = ['maçã', 'banana', 'laranja'];
for(const fruta of frutas) {
    console.log(fruta);
}

Funções

Funções tradicionais:

function soma(a, b) {
    return a + b;
}
console.log(soma(5, 3)); // 8

Arrow functions (ES6):

const soma = (a, b) => a + b;
console.log(soma(5, 3)); // 8

Arrays

Criação e manipulação:

const numeros = [10, 20, 30, 40, 50];

// Acessando elementos
console.log(numeros[0]); // 10

// Adicionando elementos
numeros.push(60);

// Métodos úteis
numeros.forEach(num => console.log(num));
const dobrados = numeros.map(num => num * 2);

Objetos

Criação e manipulação:

const pessoa = {
    nome: 'João',
    idade: 30,
    saudacao() {
        console.log(`Olá, meu nome é ${this.nome}`);
    }
};

// Acessando propriedades
console.log(pessoa.nome); // João
pessoa.saudacao(); // Olá, meu nome é João

DOM Manipulation

Interagindo com a página HTML:

// Selecionar elemento
const botao = document.querySelector('#meuBotao');

// Adicionar evento
botao.addEventListener('click', () => {
    alert('Botão clicado!');
});
