Tutorial de Python

O que é Python?

Python é uma linguagem de programação interpretada de alto nível criada por Guido van Rossum em 1991. É amplamente usada para:

  • Desenvolvimento web (Django, Flask)
  • Ciência de dados e machine learning
  • Automação de tarefas
  • Scripting
  • Desenvolvimento de jogos

Estrutura Básica de um Programa Python

# Comentário de uma linha
"""
Comentário de múltiplas linhas
"""

print("Olá, Mundo!")

Tipos de Dados Básicos

Tipo Descrição
int Números inteiros
float Números decimais
str Sequência de caracteres
bool Verdadeiro (True) ou falso (False)
list Coleção ordenada e mutável

Variáveis e Constantes

Declaração de variáveis:

nome = "João"
idade = 30
altura = 1.75
programador = True

Constantes (por convenção):

PI = 3.14159

Operadores

Operadores Aritméticos

  • + Adição
  • - Subtração
  • * Multiplicação
  • / Divisão
  • // Divisão inteira
  • % Módulo (resto da divisão)
  • ** Exponenciação

Operadores de Comparação

  • == Igual
  • != Diferente
  • > Maior que
  • < Menor que
  • >= Maior ou igual
  • <= Menor ou igual

Estruturas de Controle

Condicionais

if-elif-else:

if idade >= 18:
    print("Maior de idade")
elif idade >= 12:
    print("Adolescente")
else:
    print("Criança")

Operador ternário:

status = "Adulto" if idade >= 18 else "Menor"

Loops

for:

for i in range(10):
    print(i)

while:

i = 0
while i < 10:
    print(i)
    i += 1

Funções

Funções básicas:

def soma(a, b):
    return a + b

print(soma(5, 3))  # 8

Funções com parâmetros padrão:

def saudacao(nome="Visitante"):
    return f"Olá, {nome}!"

print(saudacao())  # Olá, Visitante!

Listas

Criação e manipulação:

numeros = [10, 20, 30, 40, 50]

# Acessando elementos
print(numeros[0])  # 10

# Adicionando elementos
numeros.append(60)

# Métodos úteis
for num in numeros:
    print(num)

dobrados = [num * 2 for num in numeros]  # List comprehension

Dicionários

Criação e manipulação:

pessoa = {
    "nome": "João",
    "idade": 30,
    "profissao": "Desenvolvedor"
}

# Acessando valores
print(pessoa["nome"])  # João

# Adicionando/alterando valores
pessoa["altura"] = 1.75

Manipulação de Arquivos

# Escrevendo em arquivo
with open("arquivo.txt", "w") as arquivo:
    arquivo.write("Olá, Python!")

# Lendo arquivo
with open("arquivo.txt", "r") as arquivo:
    conteudo = arquivo.read()
    print(conteudo)

Orientação a Objetos

class Pessoa:
    def __init__(self, nome, idade):
        self.nome = nome
        self.idade = idade

    def apresentar(self):
        print(f"Olá, meu nome é {self.nome} e tenho {self.idade} anos")

# Criando objeto
p = Pessoa("Maria", 25)
p.apresentar()
