Tutorial de C++

O que é C++?

C++ é uma linguagem de programação poderosa e versátil, criada por Bjarne Stroustrup em 1985 como uma extensão da linguagem C. É amplamente usada para:

  • Desenvolvimento de sistemas operacionais
  • Jogos
  • Aplicações de alto desempenho
  • Programação embarcada
  • Inteligência artificial

Estrutura Básica de um Programa C++

                
            #include <iostream>
using namespace std;

int main() {
    // Seu código aqui
    cout << "Olá, Mundo!" << endl;
    return 0;
}

Explicação:

  • #include <iostream> - Inclui a biblioteca para entrada/saída
  • using namespace std - Permite usar funções da biblioteca padrão sem prefixo
  • int main() - Função principal onde a execução começa
  • cout - Para imprimir na tela
  • return 0 - Indica que o programa terminou com sucesso

Tipos de Dados Básicos

Tipo Descrição
int Números inteiros
float Números decimais
double Números decimais com maior precisão
char Um único caractere
bool Verdadeiro (true) ou falso (false)

Variáveis e Constantes

Declaração de variáveis:

int idade = 25;
double altura = 1.75;
char inicial = 'J';
bool programador = true;

Constantes (valores que não mudam):

const float PI = 3.14159;
const int DIAS_SEMANA = 7;

Operadores

Operadores Aritméticos

  • + Adição
  • - Subtração
  • * Multiplicação
  • / Divisão
  • % Módulo (resto da divisão)

Operadores de Comparação

  • == Igual a
  • != Diferente de
  • > Maior que
  • < Menor que
  • >= Maior ou igual a
  • <= Menor ou igual a

Condicionais

if-else:

if (idade >= 18) {
    cout << "Maior de idade";
} else {
    cout << "Menor de idade";
}

switch-case:

switch(dia) {
    case 1: cout << "Domingo"; break;
    case 2: cout << "Segunda"; break;
    // ...
    default: cout << "Dia inválido";
}

Loops

for:

for(int i = 0; i < 10; i++) {
    cout << i << endl;
}

while:

int i = 0;
while(i < 10) {
    cout << i << endl;
    i++;
}

do-while:

int i = 0;
do {
    cout << i << endl;
    i++;
} while(i < 10);

Funções

Funções permitem organizar e reutilizar código:

// Declaração
int soma(int a, int b) {
    return a + b;
}

// Chamada
int resultado = soma(5, 3);
cout << "5 + 3 = " << resultado;

Arrays (Vetores)

Armazenam múltiplos valores do mesmo tipo:

int numeros[5] = {10, 20, 30, 40, 50};

// Acessando elementos
cout << numeros[0]; // Primeiro elemento (10)

// Alterando valores
numeros[1] = 25;

Programação Orientada a Objetos

C++ suporta os principais conceitos de OOP:

class Carro {
private:
    string marca;
    int ano;
    public:
    // Construtor
    Carro(string m, int a) {
        marca = m;
        ano = a;
    }

    // Método
    void mostrarInfo() {
        cout << "Marca: " << marca << ", Ano: " << ano;
    }
};

// Usando a classe
Carro meuCarro("Ford", 2020);
meuCarro.mostrarInfo();
