Tutorial de PHP

O que é PHP?

PHP (Hypertext Preprocessor) é uma linguagem de script server-side criada por Rasmus Lerdorf em 1994. É amplamente usada para:

  • Desenvolvimento web back-end
  • Sistemas de gerenciamento de conteúdo (WordPress, Drupal)
  • Aplicações web dinâmicas
  • Integração com bancos de dados
  • APIs RESTful

Estrutura Básica de um Script PHP

<?php
// Comentário de uma linha
/*
  Comentário de múltiplas linhas
*/

echo "Olá, Mundo!";
?>

Tipos de Dados Básicos

Tipo Descrição
integer Números inteiros
float Números decimais
string Sequência de caracteres
boolean Verdadeiro (true) ou falso (false)
array Coleção ordenada de valores

Variáveis e Constantes

Declaração de variáveis:

<?php
$nome = "João";
$idade = 30;
$altura = 1.75;
$programador = true;
?>

Constantes:

<?php
define("PI", 3.14159);
const DIAS_SEMANA = 7;
?>

Operadores

Operadores Aritméticos

  • + Adição
  • - Subtração
  • * Multiplicação
  • / Divisão
  • % Módulo (resto da divisão)
  • ** Exponenciação (PHP 5.6+)

Operadores de Comparação

  • == Igual
  • === Idêntico (valor e tipo)
  • != Diferente
  • !== Não idêntico
  • > Maior que
  • < Menor que

Estruturas de Controle

Condicionais

if-else:

<?php
if ($idade >= 18) {
    echo "Maior de idade";
} else {
    echo "Menor de idade";
}
?>

Operador ternário:

<?php
echo $idade >= 18 ? "Adulto" : "Menor";
?>

switch-case:

<?php
switch($dia) {
    case 1: echo "Domingo"; break;
    case 2: echo "Segunda"; break;
    default: echo "Dia inválido";
}
?>

Loops

for:

<?php
for($i = 0; $i < 10; $i++) {
    echo $i . "<br>";
}
?>

while:

<?php
$i = 0;
while($i < 10) {
    echo $i . "<br>";
    $i++;
}
?>

foreach (para arrays):

<?php
$frutas = ['maçã', 'banana', 'laranja'];
foreach($frutas as $fruta) {
    echo $fruta . "<br>";
}
?>

Funções

Funções básicas:

<?php
function soma($a, $b) {
    return $a + $b;
}
echo soma(5, 3); // 8
?>

Funções com parâmetros padrão:

<?php
function saudacao($nome = "Visitante") {
    return "Olá, $nome!";
}
echo saudacao(); // Olá, Visitante!
?>

Arrays

Arrays indexados e associativos:

<?php
// Array indexado
$numeros = [10, 20, 30, 40, 50];

// Array associativo
$pessoa = [
    'nome' => 'João',
    'idade' => 30,
    'profissao' => 'Desenvolvedor'
];

// Acessando elementos
echo $numeros[0]; // 10
echo $pessoa['nome']; // João
?>

Manipulação de Strings

<?php
$texto = "PHP é poderoso";

// Comprimento da string
echo strlen($texto); // 14

// Converter para maiúsculas
echo strtoupper($texto); // PHP É PODEROSO

// Substituir texto
echo str_replace("poderoso", "incrível", $texto);
?>

Conexão com Banco de Dados (MySQL)

<?php
$servername = "localhost";
$username = "usuario";
$password = "senha";
$dbname = "meubanco";

// Criar conexão
$conn = new mysqli($servername, $username, $password, $dbname);

// Verificar conexão
if ($conn->connect_error) {
    die("Falha na conexão: " . $conn->connect_error);
}

// Consulta SQL
$sql = "SELECT id, nome FROM usuarios";
$result = $conn->query($sql);

// Processar resultados
if ($result->num_rows > 0) {
    while($row = $result->fetch_assoc()) {
        echo "ID: " . $row["id"]. " - Nome: " . $row["nome"]. "<br>";
    }
} else {
    echo "0 resultados";
}

// Fechar conexão
$conn->close();
?>
